The process to solve a linear inequality is very similar to the method used to solve linear equations. The addition and subtraction properties of inequality state that the same is true for an inequality.  Read the details about linear inequality brainly.ph/question/1799662

Property of Addition and subtraction Properties of Inequality

Let a, b and c represent real numbers.

1. Addition Property of Inequality  if a< b, then a + b < b + c

2. Subtraction property of Inequality if a<b then a-c <  b - c

MULTIPLICATION AND DIVISION PROPERTIES OF INEQUALITY

Multiplying both sides of an equation by the same quantity results in an equivalent equation. However, the same is not always true for an inequality. If you multiply or divide an inequality by  a negative quantity, the direction of the inequality symbol must be reversed.

MULLTIPLICATION AND DIVISION PROPERTIES OF INEQUALITY
