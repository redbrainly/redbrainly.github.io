26. Explanation sa tagalog of resource mobilization


Ano ang Resource Mobilization?

Ang Resource Mobilization ay ang proseso ng pagkuha ng mga mapagkukunan mula sa mga resource provider gamit ang iba't ibang mekanismo upang ipatupad ang mga layunin ng isang organisasyon. Ito rin ay ang pagkuha ng mga kailangang resources sa mabisa at epektibong paraan.

Bakit Mahalaga ang Resource Mobilization? Tinitiyak ang pagpapatuloy ng paglalaan ng serbisyo ng organisasyon sa kanyang mga kliyente.  Sinusuportahan ang pagpapanatili ng organisasyon. Pinapayagan ang pagpapabuti ng mga produkto at serbisyo na kasalukuyang ibinibigay ng organisasyon. Mapanatili sa negosyo ang mgapampubliko o pribadong sektor. Binibigyang diin nito ang kakayahan ng mga indibidwal na magproseso ng mga assets at pinakikilos ang mga indibidwal tungo sa pagkamit ng mga layunin ng organisasyon.
