7. the time to complete a project varies inversely as the number of employees. if 7 people can complete a project in 10. days, how long it would it take 15 people to finish the same project


Inverse variation in mathematics problem is the representation of the relationship of two variables expressed mainly by an equation where the product of the two variables given is equal to a constant called "proportionality constant".Problems involving inverse variations are the opposite of direct variations. Thus we define direct variation as the quotient of two variables which is equal to a constant.

In solving the unknown, you should know what the given variables are.

In the problem, the values used are

1. Time - we let as T

2. Number of Employees - we let as E

The problem states that the "time to complete a project varies inversely as the number of employees"

By definition, we get the equation

[tex]k=TE[/tex] where k = proportionality constant

It says that there are 7 people that can complete a project in 10 days.

Thus, first solve for the value of k by substituting the values of T and E which are 10 days and 7 people.

[tex]k=(10)(7)[/tex]

[tex]k=70[/tex]

Since we now have the value of k, we can solve for the time it would take for 15 people to finish the same project.

[tex]k=TE\\70=T(15)\\T=\frac{70}{15}\\ T=4.67[/tex] days

Thus, the time to complete the same project if there are 15 people to work would be 4.67 days

22. 4. The time to complete a project varies inversely with the number of employees. If 3 people can complete the project in 7 days, how long will it take 5 people?​


SOLUTION :

Let the time to complete a project be y, and number of employees be x, y varies inversely with x, write as:

[tex]y = \frac{k}{x} [/tex]

Where K is the constant of Variation. If y = 7 when x = 3, solve for k.

[tex]7 = \frac{k}{3} [/tex]

[tex]k = (7)(3)[/tex]

[tex]k = 21[/tex]

Thus, If k = 21 and x = 5 , solve for y.

[tex]y = \frac{k}{x} [/tex]

[tex]y = \frac{21}{5} [/tex]

[tex]y = 4.2\:days[/tex]ANSWER :Hence, 5 people can complete the project in 4.2 days.
