The first vector is a vector component on the x-axis (Fx) and the second vector is a vector component on the y-axis (Fy)  

Using the principle rules of cosines and sines in triangles then:  

Fx = F cos α  

Fy = F sin α  

Resultan (R) :

= √Fx²+Fy²

R² = F1² + F2² + 2 F1.F2 cos α ⇒Cosine Law

the angle that F1 makes with the 50 kN(R)⇒β

21. For the figure shown below, the resultantR = 40 kN and P3 - 20 KN. Thedimensions are in mm find the value ofP, and P2​


Answer:

The value of P₁ is 58.2 kN and P₂ is 47.32 kN

Explanation:

In this problem, we are to use the concept of resultant forces to solve the requirements.

Formula to use

R = [tex]\sqrt{(\Sigma x)^2+(\Sigma y)^2}[/tex]

where

R    is the resultant force, the given unit is in kN

∑x  is the sum of all horizontal forces

∑y  is the sum of all vertical forces

Given information

R = 40 kN

P₃ = 20 kN

P₁ and P₂ = ?

Solving the problem

First, let us solve for the hypotenuse of each force.

For P₁:

r = [tex]\sqrt{300^2 + 600^2}[/tex] = 671 mm

For P₂:

r = [tex]\sqrt{300^2 + 400^2)}[/tex] = 500 mm

Now, let us solve for ∑x. Let us convert the dimensions into a meter.

∑x = P₁(0.3/0.671) + P₂(0.3/0.5) - 20 kN

Take note that (0.3/0.6) and (0.3/0.4) are all cosines of their respective forces.

∑x = 0.45P₁ + 0.6P₂ - 20 kN

0.45P₁ + 0.6P₂ - 20 = 0    

Now, let us solve for ∑y.

∑y = P₁(0.6/0.671) - P₂(0.4/0.6)

∑y = 0.89P₁ - 0.67P₂

0.89P₁ - 0.67P₂ = 0          

Now, let us substitute the horizontal and vertical forces to the formula for resultant forces

R = [tex]\sqrt{(\Sigma x)^2+(\Sigma y)^2}[/tex]

R² = ∑x² + ∑y²

40² = (0.45P₁ + 0.6P₂ - 20)² + (0.89P₁ - 0.67P₂)²       equation 1

Let us use another formula for the resultant's angle.

tan θ = ∑y / ∑x

tan 30° = (0.89P₁ - 0.67P₂) / 0.45P₁ + 0.6P₂ - 20

0.58 (0.45P₁ + 0.6P₂ - 20) = 0.89P₁ - 0.67P₂

0.26P₁ + 0.35P₂ - 11.6 = 0.89P₁ - 0.67P₂

0.26P₁ - 0.89P₁ + 0.35P₂ + 0.67P₂ = 11.6

- 0.63P₁ + 1.02P₂ = 11.6

P₁ = (11.6 - 1.02P₂) / -0.63          equation 2        

Substitute to equation 1

40² = (0.45P₁ + 0.6P₂ - 20)² + (0.89P₁ - 0.67P₂)²

1600 = [0.45(11.6 - 1.02P₂) / -0.63) +0.6P₂ - 20]² + [0.89(11.6 - 1.02P₂) / -0.63 - 0.67P₂]²

40² = [-8.29 + 0.73P₂ +0.6P₂ -20]² + [-16.39 + 1.44P₂ - 0.67P₂]²

40² = [1.33P₂ - 28.29]² + [0.77P₂ - 16.39]²

Let us some apply algebra rules, we have:

40² = 1.77P₂² - 75.25P₂ + 800.27 + 0.59P₂² - 25.24P₂ + 268.86

1600 = 2.36P₂² - 100.45P₂ + 1069.13

2.36P₂² - 100.45P₂ - 530.87 = 0

Now, let us apply quadratic equation

P₂ = 47.32 kN ; −4.75 kN

Substitute P₂ to equation 1

P₁ = (11.6 - 1.02P₂) / -0.63

P₁ = [11.6 - 1.02(47.32)] / -0.63 = 58.2 kN or

P₁ = [11.6 - 1.02(-4.75)] / -0.63 = - 10.72 kN

Now, let us have some checking:

R = [tex]\sqrt{(\Sigma x)^2+(\Sigma y)^2}[/tex]

∑x = 0.45P₁ + 0.6P₂ - 20 kN

∑y = 0.89P₁ - 0.67P₂

------------------------------------------------------------------

If P₁ = 58.2 kN and P₂ = 47.32 kN, the solve for R

∑x = 0.45P₁ + 0.6P₂ - 20 kN = 0.45(58.2) + 0.6(47.32) -20= 34.58 kN

∑y = 0.89P₁ - 0.67P₂ = 0.89(58.2) - 0.67(47.32) = 20.09 kN

R = [tex]\sqrt{(\Sigma x)^2+(\Sigma y)^2}[/tex]

R = [tex]\sqrt{(34.58)^2+(20.09)^2}[/tex]

R = 39.99 kN ≅ 40 kN

This means that we have to reject P₁ = - 10.72 kN and P₂ = - 4.75 kN

Therefore, the forces P₁ and P₂ are 58.2 kN and 47.32 kN respectively.
