170cm ang sagot, kasi 90 percent ng height ng lalaki yung babae.

Answer:

Keith's actual height is 170 cm.

Step-by-step explanation:

Ratio:

Scaled height of Ezra is to scaled height of Keith (in photograph)

⇒  9 : 10

Actual height:

Ezra ⇒ 153 cm

Keith ⇒  y

Find the actual height of Keith "y" using direct proportion:

9 : 10 = 153 : y

9y = (10) (153)

9y/9 = 1,530/9

y = 170 cm

Keith's actual height is 170 cm.

30. Danny and Esther are Cathy's children. Esther is 5 years younger than Danny and Cathy is three times Danny's age. If Cathy's age 30 years ago was the same as Danny's age now, how old is Esther?


answer: Esther is 10 years old

given:

E=D-5

C=3D

C-30=D

formula: E=(C-30)-5 {{{substitution}}}

find the value for D (substitution again):

3D-30=D

2D=30

D=15

Now find the value for c using the value for D:

C=3D

C=3(15)

C=45


substitute the found value to the original formula:

E=(C-30)-5

E=(45-30)-5

E=15-5

E=10

Answer is 10
