14. The cost of the renting room is Php 300 a day plus Php 100 per person. If the room is rented for d days and include m persons. Represent the cost C in terms of d and m.​ ​


Answer:

C = 300d + 100m

Step-by-step explanation:

Let d be the number of days

    m be the number of persons

    C be the total cost

We have a given Php 300 per day which is equivalent to :

                                            300d

And a given of Php 100 per person which is equivalent to :

                                            100m

So the Total Cost  C = 300d + 100m
