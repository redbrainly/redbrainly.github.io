Answer:

Microsoft Paint or MS Paint (MSP) is a legacy graphics drawing program that has been shipped with all versions of Windows. MSP is an easy way to create different kinds of graphics, with tools such as brushes, erasers, pens and shape generators.

The program opens and saves files in Windows bitmap (BMP), JPEG, GIF, PNG, and single-page TIFF formats.
