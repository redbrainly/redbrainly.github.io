Primary Source is an original copy, resource, recording, data, etc. meaning it is not plagiarised, copied, or inspired by any other resource.

Examples:

InterviewLetters and MessagesJournalDiary

While Secondary Source is a copy, resource, recording, data, etc. that was inspired and made using the primary source. Secondary source will not be made if the primary source did not exist.

Examples:
